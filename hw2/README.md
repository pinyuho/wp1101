# HW2 Spec




### HW1 Descriptions

- Cover
    - 網頁名稱 (my photo album)
    - 下拉提示動畫 (scroll down)

- Albums
    - 相簿選擇區塊: 滑鼠移動到上面，則圖片會變模糊並且出現相簿名稱跟敘述、滑鼠游標變為pointer

- Photos
    - 相片預覽圖 (⼩圖): 滑鼠移動到上面，則圖片會變透明、放大、滑鼠游標變為pointer
    - 相片展⽰窗格 (⼤圖): 滑鼠移動到上面，則圖片會有陰影、放大、以及左右搖動的動畫

### Additional Feature

- RWD

    - Photo section: 畫面縮小則四格圖片會變為兩格
    - Album section: 畫面縮小則第二個欄位會跑到下一行
